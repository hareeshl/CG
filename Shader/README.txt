Extra credit 
------------

1.Arbitrarily sized images

An additional text file named window.txt is provided. This contains the interface window width and
height. The format of the file is as follows - <windowwidth><space><windowheight>

2.Support arbitratily viewing setup

An additional file named view.txt is provided which specifies the eye's location, view up vector and
lookat vector. The format the file is as follows - <eyex><space><eyey><space><eyez><space><viewupx>
<space><viewupy><space><viewupz><space><lookatx><space><lookatyy><space><lookatz>

3.Texture mapping

Used texture mapping and blended lighting and texture colors.Press 'Space bar' to rotate the location
of the light
